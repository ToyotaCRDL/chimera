# <b>Habitat</b>

Habitat-Sim is a high-performance physics-enabled 3D simulator.

This class provides the interface to the Habitat-Sim (and Habitat-Lab).
Prease see the details of Habitat-Sim at the following link.

[https://github.com/facebookresearch/habitat-sim](https://github.com/facebookresearch/habitat-sim)

[https://github.com/facebookresearch/habitat-lab](https://github.com/facebookresearch/habitat-lab)
